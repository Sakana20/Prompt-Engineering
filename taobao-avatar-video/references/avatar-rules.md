# 固定 IP 数字人口播 Prompt 规则

根据已通过校验的口播文案生成一段完整、自然、连贯的视频 Prompt。文案是场景、人物
状态、情绪、商品使用方式和表达语气的唯一语义依据。不要复述文案，不生成脚本或分镜。

## 固定账号 IP

- 22–26 岁亚洲女生，邻家女孩气质，真实自然、亲和、干净、生活化；
- 分享日常购物、生活方式和真实消费体验；
- 不是测评人、带货主播、主持人或销售；
- 保持人物风格与脸部特征一致；服装、发型、场景、动作和情绪随文案调整；
- 禁止网红脸、明星脸、欧美妆、浓妆、夸张饰品、品牌 Logo 和塑料皮肤。

文字提示只能约束人物风格。严格同脸必须使用用户确认的参考图或复用人物素材。

## 场景与商品

- 优先由文案第一句判断场景，整个视频只使用一个符合逻辑的生活场景；
- 商品自然位于桌面、餐桌、台面、办公桌、购物袋、沙发、玄关或手边；
- 需要时自然拿起，不全程手持，不持续展示包装，不刻意展示 Logo；
- 商品是生活的一部分，不使用商业广告棚或直播间陈列。

## 表演与镜头

- 面对镜头自然口播，眼神自然，口型同步，节奏自然；
- 允许自然微笑、眨眼、轻微点头、少量手势和轻微重心变化；
- 禁止主持、演讲、直播带货动作，禁止夸张动作、频繁挥手和背稿感；
- 竖屏 9:16，固定机位，中景或半身，人物居中，一镜到底；
- 不切镜、不运镜、不推拉摇移，模拟手机固定拍摄和真实生活 Vlog。

## 画面

真实摄影、手机实拍、生活记录、自然光、暖色调、真实曝光与白平衡、真实肤色、轻微背景
虚化。保留皮肤纹理，不过度磨皮、锐化或美颜。可描述 HDR、8K 和电影级真实摄影，但
整体不得产生商业摆拍或明显 AI 感。

禁止提词器、字幕、贴纸、特效、品牌 Logo、动漫、二次元、CG、AI 主播、虚拟主播和
明显 AI 生成痕迹。

最终只输出数字人 Prompt 正文，不输出标题、解释、Markdown 或额外提示。
